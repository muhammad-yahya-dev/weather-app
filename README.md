# 🌤️ Modern Weather Dashboard

A sleek, responsive, and real-time Weather Web Application that fetches live atmospheric data for any city globally. This project demonstrates dynamic API integration, asynchronous JavaScript, and elegant UI design.

---

## 🚀 Features

| Feature | Description |
| :--- | :--- |
| **🔍 Live Search** | Users can input any global city name to immediately fetch current data. |
| **🌡️ Real-Time Temperature** | Displays accurate, up-to-the-minute local temperatures dynamically. |
| **💨 Atmospheric Metrics** | Shows detailed weather insights including Humidity percentages and Wind Speed. |
| **📱 Fully Responsive UI** | Crafted with an eye-catching gradient aesthetic that adapts perfectly to desktop and mobile screens. |

---

## 🛠️ Tech Stack & Concepts Used

*   **Structure & Styling:** Semantic HTML5 and Modern CSS3 (Flexbox, gradients, and custom layouts).
*   **Core Logic:** Vanilla JavaScript (ES6+).
*   **Data Fetching:** Asynchronous operations using the Fetch API to securely request live third-party weather data.
*   **DOM Manipulation:** Dynamically updating text elements and weather statistics without reloading the web page.
